R.O.V.E.R - Sistema de Panel de Alimentación y Seguimiento Solar
Código Original (Arduino) y Simulación en Python

Este repositorio reúne el código desarrollado para el sistema de orientación solar del proyecto R.O.V.E.R, específicamente el módulo encargado del panel de alimentación.
El propósito principal de este sistema es permitir que el panel solar ajuste su posición automáticamente para maximizar la captación de energía a partir de la distribución de luz detectada por cuatro sensores fotoresistivos (LDR).

El repositorio contiene tanto el código original implementado en Arduino, utilizado físicamente en el proyecto, como una simulación completa realizada en Python(Jupyter), diseñada para reproducir el funcionamiento del sistema mediante Programación Orientada a Objetos (POO).
